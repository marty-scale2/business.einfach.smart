So nutzt du diesen Skill

In 4 Schritten startklar.

Der KI Floskel-Vermeider Skill ist in wenigen Minuten eingerichtet, egal ob du Claude im Browser oder als Desktop-App nutzt. Danach scannt Claude deinen Text auf klassische KI-Muster und liefert dir sofort die menschlichere Alternative.

Was dieser Skill macht

Er erkennt alle 13 klassischen KI-Floskeln in deinem Text, erklärt warum sie beim Leser auffliegen und liefert direkt die bessere Formulierung. Kein „Die Wahrheit ist..." mehr. Kein Stakkato-Dreier. Kein KI-Geruch.

Datei herunterladen

Du hast nach der Eintragung einen Link zu einem Google Drive Ordner erhalten. Öffne den Link, klicke auf „Herunterladen" und speichere die .skill-Datei auf deinem Computer.

💡
Kein Google-Account nötig
Du brauchst keinen Google-Account zum Herunterladen. Der Link funktioniert direkt im Browser.

.skill-Datei ablegen – je nach Version

Jetzt kommt es darauf an, welche Version von Claude du nutzt. Wähle die Option, die auf dich zutrifft.

A
Claude Code (Desktop / Terminal)
Lege die .skill-Datei in deinem Claude-Arbeitsbereich ab, unter .claude → skills. Der Pfad sieht dann so aus: .claude/skills/ki-floskel-vermeider
B
Browser-Version (claude.ai)
Die .skill-Datei direkt so hochladen, wie sie ist.
⚠️

Nicht sicher, welche Version du nutzt? Wenn du Claude über den Browser auf claude.ai öffnest, bist du in der Browser-Version. Claude Code läuft im Terminal oder als Desktop-App.

Skill in claude.ai hochladen

Dieser Schritt gilt nur für die Browser-Version. Claude Code-Nutzer können direkt zu Schritt 4 springen.

1
Zu „Anpassen" navigieren
Öffne claude.ai und klicke links im Menü auf „Anpassen". Dann auf den Tab „Skills".
2
„Skill erstellen" öffnen
Klicke oben rechts auf das + Symbol und dann auf „Skill erstellen".
3
.skill-Datei hochladen
Wähle „Skill hochladen" und lade die .skill-Datei direkt hoch. Einfach so wie sie ist reinziehen.
Anpassen → Skills → + → Skill erstellen → Skill hochladen

Neuen Chat starten mit „/"

Starte einen neuen Chat. Tippe / ins Eingabefeld. Dir wird sofort der neue Skill „ki-floskel-vermeider" angezeigt. Wähle ihn aus. Fertig.

Claude erkennt jetzt klassische KI-Floskeln in deinem Text und schlägt direkt menschlichere Formulierungen vor. Kein „Die Wahrheit ist..." mehr, kein KI-Geruch.
